# 📊 Expense Management Dashboard using Microsoft Excel

An interactive **Expense Management Dashboard** built using **Microsoft Excel** to analyze and monitor organizational expenses. This dashboard provides key financial insights through KPI cards, Pivot Tables, Pivot Charts, and Slicers.

## 🚀 Project Features

- 📌 Total Expenses KPI
- 💰 Budget Allocated
- 💵 Budget Remaining
- 📈 Monthly Expense Trend
- 🏢 Department-wise Expense Analysis
- 📂 Category-wise Expense Analysis
- 💳 Payment Method Analysis
- ✅ Approval Status Summary
- 🎯 Interactive Slicers for Filtering

---

## 🛠 Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Validation

## 📂 Dataset Information

The dataset contains:

- Expense ID
- Date
- Department
- Category
- Description
- Amount (INR)
- Payment Method
- Approval Status
- Budget Allocated
- Budget Remaining
- Vendor Name
- Invoice Number
- Month

---

## 📈 Key Insights

- Monitor overall organizational expenses.
- Compare budget allocation with actual spending.
- Analyze department-wise expenditure.
- Identify the most used payment methods.
- Track monthly expense trends.
- Monitor approval status of expenses.
